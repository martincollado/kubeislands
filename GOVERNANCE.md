# Governance

KubeIslands is maintained by [@martincollado](https://github.com/martincollado) (founding maintainer).

## Adding Maintainers

New maintainers are added by existing maintainer consensus based on sustained, high-quality contribution over at least 3 merged PRs.

## Decision Making

Decisions about direction, API changes, and release cadence happen in GitHub Issues tagged `type/proposal`. Comments are open for 7 days before a call is made.

Breaking changes require a `BREAKING CHANGE:` footer in the commit and at least one maintainer +1.

## Release Cadence

No fixed cadence for v0.x. Releases happen when meaningful changes accumulate. Patch releases for security fixes are made as needed.
